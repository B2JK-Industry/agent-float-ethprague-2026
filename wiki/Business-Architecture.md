# Business Architecture

## Business Category

Upgrade Siren is contract-risk intelligence for upgradeable Ethereum systems.

## Buyer Logic

| Buyer | Why they care |
|---|---|
| DAO | Avoid approving dangerous upgrades |
| Delegate | Get fast evidence before voting |
| Wallet | Warn users before interaction |
| Explorer | Add plain-language upgrade risk |
| Launch platform | Screen ventures before funding |
| Fund | Monitor portfolio contract risk |

## Product Packaging

Free:

- public lookup
- basic verdict
- report link
- governance comment

Paid:

- watchlists
- signed reports
- API
- wallet/explorer integrations
- venture due-diligence monitoring

## Umia Fit

If used with Umia, Siren Agent becomes a due-diligence layer:

1. Check venture contracts before funding.
2. Warn if contracts are unverified or upgradeable by a weak admin.
3. Monitor post-launch implementation changes.
4. Produce signed reports for investor confidence.

This complements Umia. It does not replace Umia's funding or venture mechanics.
