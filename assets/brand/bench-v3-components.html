STATES · color tracks the source state

×0.85

DISCOUNTED · 0.20–0.40
×0.30

MISSING · ×0.00
×0.00
Source did not return. Dashed border — the colorblind-safe identifier.
INVALID
INVALID
Active forgery. Siren. The only pill with tinted fill.
Do
  • Pin one pill to every source row in the Trust Ledger. No source ships unweighted.
  • Always two decimals. ×0.40, not ×.4 or ×0.4.
  • Use the dashed border for missing — it survives every CB profile.
Don't
  • Don't use percentages (40%) — multiplier semantics are clearer.
  • Don't display ×1.00 as ×1; alignment breaks across rows.
  • Don't mix INVALID and ×0.00 — silence and lie are different states.

STATES · 5 kinds

The forward action. One per surface.

Do

  • Use REFUSE only for destructive or refusal actions — not for deletes of harmless items.
  • Always wire :focus-visible with the 3px accent ring.
Don't
  • Don't round corners. The system has one radius: zero.
  • Don't add icons inside the label. Use → ← arrows or external glyphs only.
  • Don't use a filled siren button as a "delete" — it's the contract refusal, not a CRUD op.
