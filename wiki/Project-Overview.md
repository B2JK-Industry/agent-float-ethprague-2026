# Project Overview

## What Is Agent Float?

Agent Float is a capital market interface for working AI agents. It lets builders present agents that already perform paid work, prove that work through on-chain receipts, and route fundraising through Umia's venture infrastructure.

The product is intentionally narrow:

- It does not run agents.
- It does not sell prompts.
- It does not replace Umia.
- It does not invent a custom legal/token model.
- It does not fund agents without proof of work.

The core rule is:

> No receipts, no float.

## Why It Exists

AI agents can already perform useful work: grant research, monitoring, scraping, summarization, alerts, procurement analysis, and operational tasks. But funding them is awkward.

Builders need capital for:

- compute,
- API credits,
- data sources,
- distribution,
- reliability upgrades,
- better agent tooling.

Investors need evidence that an agent is more than a landing page. Agent Float gives investors:

- an ENS identity,
- a public receipts feed,
- visible revenue activity,
- milestone commitments,
- builder collateral,
- a Umia-native funding route.
